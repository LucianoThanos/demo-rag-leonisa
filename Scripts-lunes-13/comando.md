### 🛠️ Comandos de Configuración Inicial (Para los alumnos)

**Paso 1: Iniciar el proyecto de Node (rápido)**

**Bash**

```
npm init -y
```

**Paso 2: Instalar todas las dependencias necesarias**

**Bash**

```
npm install dotenv openai @google/generative-ai
```

acuérdense que como instalamos `dotenv`, tienen que crear un archivo que se llame exactamente `.env`
